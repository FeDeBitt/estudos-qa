# estudos-qa
Repositório de práticas em QA. Contém análises de requisitos, casos de teste e gestão de defeitos usando Jira e TestRail.

# Portfólio de Quality Assurance

Olá! Criei este repositório para organizar e compartilhar a minha evolução prática na área de testes e qualidade de software. Como estudante de Engenharia de Software, busco sempre conectar a base teórica com a aplicação no mundo real.

Neste espaço, você não vai encontrar apenas código, mas sim o raciocínio analítico necessário para garantir a qualidade de um produto. O foco do projeto é demonstrar clareza na documentação, atenção aos detalhes e o entendimento de todo o ciclo de vida de testes.

## O que tem neste repositório:
* Mapeamento de características de qualidade baseadas em normas ISO
* Decomposição de features e análise de requisitos
* Planejamento e casos de teste estruturados
* Relatórios de bugs detalhados
* Aplicação de práticas utilizando ferramentas de mercado (Jira e TestRail)
